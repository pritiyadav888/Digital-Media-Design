What is Digital Media Design?

Digital Media Design is a dynamic field that combines creativity and technology to communicate ideas and information through digital channels. It's about crafting engaging experiences that captivate audiences in the digital world.

Digital media design encompasses a wide range of specializations:

  • Web Design: Creating websites and web applications.
  • UI/UX Design: Focusing on user interface and user experience.
  • Graphic Design: Creating visual assets for digital platforms.
  • Animation: Creating moving images and visual effects.
  • Video Production: Filming, editing, and producing video content.
  • Interactive Media: Designing interactive experiences like games or simulations.

Each type plays a vital role in shaping how we interact with digital content.

Digital media design is used across many industries and contexts:

  • Marketing & Advertising: Creating campaigns, websites, and social media content.
  • Education: Designing online courses, interactive learning tools.
  • Entertainment: Developing video games, movies, and streaming content.
  • Information & Communication: Building news websites, social platforms, and communication apps.

Digital media is all around us, shaping how we communicate, learn, and experience the world.

HTML Basics: Structuring Your Web Content

HTML (HyperText Markup Language) is the foundation of every webpage. It provides the structure and meaning to your content, like headings, paragraphs, lists, and images.

HTML elements are the core components of a webpage. They consist of:

  • **Opening tag:** Like `<p>` to start a paragraph.
  • **Content:** The actual text, images, or other elements.
  • **Closing tag:** Like `</p>` to end a paragraph.

Browsers interpret these tags to display the content correctly. For example, `<h1>` tells the browser "This is the main heading."

  • `<html>`: The root element, containing the entire HTML document.
  • `<head>`: Contains information about the page (like its title, which appears in the browser tab).
  • `<body>`: Holds the visible content of the page.
  • `<h1>` to `<h6>`: Headings of different sizes.
  • `<p>`: Paragraphs of text.
  • `<a href="...">`: Links to other pages or sections.
  • `<img src="...">`: Images.
  • `<div>`: A generic container for grouping content.
  • `<span>`: A generic inline container for styling parts of text.

CSS Basics: Styling Your Webpages

CSS (Cascading Style Sheets) controls the visual presentation of your HTML content. It's how you define colors, fonts, layout, and responsiveness.

CSS rules have a simple structure:

selector {
  property: value;
}
  • **Selector:** Targets the HTML element(s) you want to style (e.g., `p` for paragraphs, `h1` for headings).
  • **Property:** The aspect of the element you want to change (e.g., `color`, `font-size`).
  • **Value:** The setting for that property (e.g., `red`, `16px`).

CSS properties are applied to HTML elements based on their selectors.

  • `color`: Text color.
  • `background-color`: Background color of an element.
  • `font-family`: Font style (e.g., Arial, Times New Roman).
  • `font-size`: Text size.
  • `font-weight`: Boldness of text.
  • `margin`: Space outside an element.
  • `padding`: Space inside an element.
  • `display`: How an element is displayed (e.g., `block`, `inline`, `flex`, `grid`).

How Does Browser See Them?

Browsers have a specific way of interpreting and displaying HTML and CSS. Let's explore the key concepts:

The browser uses a **rendering engine** to convert your HTML and CSS code into the visual webpage you see. Different browsers use different engines (e.g., Blink, Gecko, WebKit), which can sometimes lead to slight variations in how pages are displayed.

The DOM is a tree-like representation of your HTML structure. The browser uses the DOM to understand the relationships between different elements and how to style them. JavaScript also uses the DOM to manipulate webpage content.

Every HTML element is treated as a rectangular box. The box model describes the different layers that make up this box:

  • **Content:** The actual text, image, or other media within the element.
  • **Padding:** Space *inside* the element, between the content and the border.
  • **Border:** A line surrounding the element.
  • **Margin:** Space *outside* the element, pushing it away from other elements.

Week 1 Summary

This week, we've laid the groundwork for web design by exploring the essential technologies of HTML and CSS. You've learned how HTML provides structure and meaning, and how CSS controls the visual presentation.

  • HTML is for structure and content.
  • CSS is for styling and layout.
  • Semantic HTML makes your content meaningful.
  • CSS selectors target specific HTML elements.
  • CSS Grid is a powerful layout tool.
